操作說明
• 點擊物件 → 選取（變紅）
• Shift + 點擊 → 多選/取消多選
• W A S D → 前 左 後 右（半格）
• R F → 上 下（F 不低於地面）
• Q → 新增方塊並選取
• G → 新增地板
• H → 生成牆壁
• L → 往右（X+）複製選中物件（可多選）
• K → 向後（Z+）複製選中物件（可多選）
• J → 往左（X-）複製選中物件（可多選）
• I → 向前（Z-）複製選中物件（可多選）
• O → 向上（Y+）複製選中物件（可多選）
• Z → 還原上一步
• X → 退回下一步
• V → 刪除所有選中物件
• P → 整層複製
• Delete / Backspace → 刪除最後一個
• 上下/左右翻轉 90° → 只作用於單選
• 顯示模式：不透明 → 半透明 → 純線框
• 1 → 隱藏/顯示介面　　2 → 下載 PNG　　3 → 儲存 JSON　　4 → 載入 JSON
新物件使用此顏色；選中物件可即時改變
